## Django
---
### 环境配置
- 安装`Django`
```python
pip3.10 install django    // 最好指定Python的版本
```
需要将`Python`加入环境变量path中
在终端中输入`django-admin`,出现提示即可
## 创建项目
> django-admin startproject  su7

- 项目文件目录
- App

> **创建app:** python manage.py startapp app01

**所以最终一共有两个主要的文件目录：一个是默认的文件，一个是项目生成时创建的文件。**
